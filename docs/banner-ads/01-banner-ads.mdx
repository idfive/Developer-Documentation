# Banner Ads

## HTML Structure
This structure is a boilerplate that can be used to create a new ad. 

See highlighted comments calling out unique aspects of the ad:

```HTML showLineNumbers
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    
    <!-- ad.size meta tag with correct sizing for ad dimensions -->
    <meta name="ad.size" content="width=160,height=600" />
    <title>Banner Ad Title - 160x600</title>
    
    <!-- link to external font (can be Google, Adobe Typekit, etc) -->
    <link rel="preconnect" href="https://fonts.googleapis.com" />
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
    <link href="https://fonts.googleapis.com/css2?family=font" rel="stylesheet"/>
    
    <!-- link to external css file -->
    <link rel="stylesheet" href="css/style.css" />
    
    <!-- click tag -->
    <script type="text/javascript">
      var clickTag = "https://www.google.com/";
    </script>
  </head>
  <body>
    
    <!-- container for ad -->
    <div class="banner-ad">
      
      <!-- apply click tag in a link covering the ad -->
      <a href="javascript:window.open(window.clickTag)" class="banner-ad__contact"></a>
      
      <!-- container for content -->
      <div class="banner-ad__content">
        
        <!-- content that will not change with "slides" can go here -->
        <div class="banner-ad__logo">
          <img src="assets/logo.svg" width="140" height="27" alt="Logo Alt Text" />
        </div>
        <div class="button">Button Text</div>
        
        <!-- slide 1 -->
        <div class="banner-ad__slide banner-ad__slide-1">
          <span>Slide 1 Text Content</span>
        </div>
        
        <!-- slide 2 -->
        <div class="banner-ad__slide banner-ad__slide-2">
          <span>Slide 2 Text Content</span>
        </div>
      </div>
    </div>
    
    <!-- link to external js file -->
    <script src="js/script.js"></script>
  </body>
</html>

```

## Clicktags

Clicktags are used by Ad Networks to track clicks on the ads being served out. In the HTML file for the ad:

```HTML
  <script type="text/javascript">
    var clickTag = "https://www.google.com/";
  </script>
</head>
<body>
  <div class="banner-ad">
    <a href="javascript:window.open(window.clickTag)" class="banner-ad__contact"></a>
    // Rest of ad markup
```

- In this example the `https://www.google.com/` would be replaced by the desired landing page URL where the ad should link, typically including trakcing parameters provided by the ad network.
- The `clickTag` variable is setting the url.
- The `<a>` tag covers the entire ad providing the click through.

## Javascript
This javascript can be used as is. The only variables that may need to change are `this.totalCycles` and `this.cycleDuration` depending on the ad network specs. In this example the ad animations will run three times with each cycle lasting 15 seconds, for a total of 45 seconds. The CSS animations are set up to last for a total of 15 seconds.

```js showLineNumbers
// Banner ad animation controller
class BannerAnimationController {
  constructor() {
    this.currentCycle = 0;
    this.totalCycles = 3; // run the ad 3 times
    this.cycleDuration = 15000; // 15 seconds per cycle
    this.animatedElements = [];

    this.init();
  }

  init() {

    // Start the animation cycles
    this.startAnimationCycle();
  }

  startAnimationCycle() {
    if (this.currentCycle >= this.totalCycles) {
      return; // Stop after 3 cycles
    }

    // Reset all animations by removing and re-adding animation classes
    this.resetAnimations();

    // Start the next cycle
    setTimeout(() => {
      this.currentCycle++;
      this.startAnimationCycle();
    }, this.cycleDuration);
  }

  resetAnimations() {
    // Force reflow to restart CSS animations
    const bannerAd = document.querySelector(".banner-ad");
    const originalDisplay = bannerAd.style.display;
    bannerAd.style.display = "none";
    bannerAd.offsetHeight; // Trigger reflow
    bannerAd.style.display = originalDisplay;
  }
}

// Initialize the animation controller when the DOM is loaded
document.addEventListener("DOMContentLoaded", () => {
  new BannerAnimationController();
});

```
